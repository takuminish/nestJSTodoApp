# NestJS練習用 TODOアプリケーション(WebAPI)

## 概要
- NestJSの練習用としてTODOアプリケーション(WebAPI)を作成する。
- RestAPIに乗っ取ってAPI設計を行う。
- DBはMySQLを使用する。

- Docker
- MySQL

## 起動方法
```
$ docker-compose up -d
```
`localhost:3000/todo`にアクセスして、TODOテーブルのレコードが取得できていれば良い。
